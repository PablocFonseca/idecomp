Operação dos Reservatórios Equivalentes de Energia (dec_oper_ree.csv)

As informações de operação dos reservatórios equivalentes de energia são armazenadas na classe:

class DecOperRee(data=<cfinterface.data.blockdata.BlockData object>)[código fonte]

Arquivo com a operação por REE do DECOMP.

property tabela: Optional[DataFrame]

A tabela de dados que está contida no arquivo.

  • periodo (int)

  • no (int)

  • cenario (int)

  • indiceRee (int)

  • nomeRee (str)

  • indiceSubmercado (int)

  • nomeSubmercado (str)

  • enaMWmes (float)

  • earmInicialMWmes (float)

  • earmInicialPercentual (float)

  • earmFinalMWmes (float)

  • earmFinalPercentual (float)

  • earmMaximoMWmes (float)

Retorna

A tabela como um dataframe

Tipo de retorno

pd.DataFrame | None
