Dados gerais (dadger.rvX)¶

Visão geral do modelo¶

As informações de entrada gerais do DECOMP, localizadas no arquivo geralmente denominado dadger.rvX, onde X varia de 0 a 4, são armazenadas na classe:

class Dadger(dados: idecomp._utils.dadosdadger.DadosDadger)[código fonte]¶

Armazena os dados de entrada do DECOMP

Esta classe lida com as informações de entrada fornecidas ao DECOMP no dadger.rvx.
