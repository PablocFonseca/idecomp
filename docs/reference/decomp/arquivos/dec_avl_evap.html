Avaliação da Evaporação Linear (dec_avl_evap.csv)

A avaliação da modelagem por evaporação linear é armazenada na classe:

class DecAvlEvap(data=<cfinterface.data.blockdata.BlockData object>)[código fonte]

Arquivo com a avaliação da evaporação linear do DECOMP.

property tabela: Optional[DataFrame]

A tabela de dados que está contida no arquivo.

  • periodo (int)

  • indiceUsina (int)

  • nomeUsina (str)

  • submercado (int)

  • ree (int)

  • volumeArmazenadoHm3 (float)

  • evaporacaoCalculadaHm3 (float)

  • evaporacaoModeloHm3 (float)

  • desvioAbsolutoHm3 (float)

  • desvioPercentual (float)

Retorna

A tabela como um dataframe

Tipo de retorno

pd.DataFrame | None
