Eco da Discretização Temporal (dec_eco_discr.csv)

O eco da discretização temporal do estudo é armazenado na classe:

class DecEcoDiscr(data=<cfinterface.data.blockdata.BlockData object>)[código fonte]

Arquivo com a discretização temporal dos estágios e patamares do DECOMP.

property tabela: Optional[DataFrame]

A tabela de dados que está contida no arquivo.

  • periodo (int)

  • patamar (int)

  • duracao (float)

  • numeroPatamares (int)

  • numeroAberturas (int)

Retorna

A tabela como um dataframe

Tipo de retorno

pd.DataFrame | None
