vazoes.dat¶

O primeiro passo para realizar o processamento do arquivo, assim como os demais arquivos de saída, é a leitura. A função de leitura recebe dois argumentos, sendo que o segundo é opcional.

from idecomp.decomp import Vazoes

arq = Vazoes.le_arquivo(".", nome_arquivo="vazoes.rv0")

O arquivo de vazões contém informações do histórico de vazões por posto de medição, servindo de entrada para os modelos de planejamento.

df = arq.vazoes
print(df)
